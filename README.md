# Banco de Dados - SQL Loja

- Projeto desenvolvido durante meus estudos de Banco de Dados.


## Objetivo

- Desenvolver um banco de dados relacional para gerenciar clientes, produtos, pedidos e itens de pedido, aplicando os principais conceitos de modelagem e SQL.

- Aplicação de conceitos:

    - Iniciar um novo banco de dados com CREATE DATABASE

    - Criar tabelas com CREATE TABLE

    - Definir PRIMARY KEY

    - Definir FOREIGN KEY

    - Inserir dados com INSERT

    - Consultar dados com SELECT

    - Consultas utilizando JOIN entre tabelas

    - Versionamento do projeto utilizando Git e publicação no GitHub


## Tecnologias:

- SQL
- MySQL
- Git
- GitHub

## Estrutura do Banco

Tabelas
│
├── Clientes (armazena os dados dos clientes)
├── Produtos (armazena os produtos disponíveis)
├── Pedidos (registra os pedidos realizados)
└── Itens_pedidos (relaciona produtos aos pedidos)


## Como executar

1. Execute o arquivo `script.sql` para criar o banco e as tabelas.
2. Execute o arquivo `inserts.sql` para inserir os dados de exemplo.
3. Execute o arquivo `consultas.sql` para testar as consultas.
