# Sistema de Loja

Projeto desenvolvido durante meus estudos de Banco de Dados.

## Objetivo

Criar um banco de dados para uma loja para controlar clientes, produtos, pedidos e itens de pedido.

Aplicação de conceitos:

✅ Iniciar um novo banco de dados com CREATE DATABASE

✅ Criar tabelas com CREATE TABLE

✅ Definir PRIMARY KEY

✅ Definir FOREIGN KEY

✅ Inserir dados com INSERT

✅ Consultar dados com SELECT

✅ Fazer um JOIN

✅ Trabalhar com repositório local usando Git e repositório remoto com Git Hub





## Tecnologias:

- SQL
- MySQL
- Git e GitHub
